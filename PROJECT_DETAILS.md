## Project Scope

**MVP Features (Must-Have):**
- ✅ User login (2 demo accounts: student1/student123, student2/student123)
- ✅ Course selection from 10 predefined courses
- ✅ Confirmation summary page
- ✅ Responsive UI for desktop and mobile

**NOT Included (Realistic Scope):**
- ❌ Payment system
- ❌ Admin dashboard
- ❌ Email notifications
- ❌ Real-time database

## Technology Stack

- **Frontend:** HTML, CSS, JavaScript (Bootstrap 5)
- **Backend:** Node.js, Express (Note: Current implementation uses client-side only with sessionStorage)
- **Database:** JSON file / SessionStorage (mock data)
- **Tools:** GitHub, Figma, Trello
